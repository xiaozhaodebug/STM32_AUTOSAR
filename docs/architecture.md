## 模块说明

### 1. 微控制器驱动层 (MCAL)

#### CanDriver
- 文件：`src/drivers/CanDriver.c`
- 功能：CAN1 硬件驱动
- 特性：
  - 500Kbps 波特率
  - 中断接收
  - 过滤器支持

#### DebugLog
- 文件：`src/utils/DebugLog.c`
- 功能：调试日志输出
- 接口：USART1 115200bps

### 2. 抽象层 (Abstraction)

#### CanIf
- 文件：`AUTOSAR/CanIf/CanIf.c`
- 功能：CAN 接口抽象
- 提供统一的 CAN 访问接口

#### PduR
- 文件：`AUTOSAR/PduR/PduR.c`
- 功能：PDU 路由
- 将接收到的 PDU 路由到上层模块

### 3. 服务层 (Services)

#### Com
- 文件：`AUTOSAR/Com/Com.c`
- 功能：通信服务
- 信号路由和打包解包

#### Dcm
- 文件：`AUTOSAR/Dcm/Dcm.c`
- 功能：诊断通信管理
- UDS 服务实现

#### EcuM
- 文件：`AUTOSAR/EcuM/EcuM.c`
- 功能：ECU 状态管理
- 初始化序列和状态机

## 数据流

### CAN 发送流程

```
应用层
   │
   ▼
Com_SendSignal() ──► Com 模块
   │
   ▼
PduR_Transmit() ───► PduR 模块
   │
   ▼
CanIf_Transmit() ──► CanIf 模块
   │
   ▼
CanDriver_Send() ──► CAN 硬件
```

### CAN 接收流程

```
CAN 硬件 (中断)
   │
   ▼
CAN1_RX0_IRQHandler()
   │
   ▼
CanIf_RxIndication() ──► CanIf 模块
   │
   ▼
PduR_RxIndication() ───► PduR 模块
   │
   ▼
Com_RxIndication() ────► Com 模块
   │
   ▼
应用层回调
```

## 任务调度

### 周期性任务

| 周期 | 任务 | 模块 | 功能 |
|------|------|------|------|
| 1ms  | Dcm_MainFunction() | Dcm | 诊断处理 |
| 5ms  | EcuM_MainFunction() | EcuM | 状态管理 |
| 10ms | Com_MainFunctionRx() | Com | 接收处理 |
| 10ms | Com_MainFunctionTx() | Com | 发送处理 |
| 100ms| Com_MainFunctionRouteSignals() | Com | 信号路由 |

### 中断处理

| 中断源 | 优先级 | 处理函数 | 功能 |
|--------|--------|----------|------|
| SysTick | 高 | SysTick_Handler() | 系统时钟 |
| CAN1_RX0 | 中 | CAN1_RX0_IRQHandler() | CAN接收 |

## 内存布局

### Flash 布局

```
0x0800_0000 ├─ 启动代码 (startup)
            │
0x0800_2000 ├─ 中断向量表
            │
0x0800_3000 ├─ 代码段 (.text)
            │   - 驱动代码
            │   - AUTOSAR 模块
            │   - 应用代码
            │
0x0801_0000 ├─ 常量数据 (.rodata)
            │
0x0801_F000 └─ 配置数据
```

### RAM 布局

```
0x2000_0000 ├─ 向量表重映射
            │
0x2000_0100 ├─ 栈空间 (Stack)
            │   (向下增长)
            │
0x2000_8000 ├─ 全局变量 (.data/.bss)
            │   - AUTOSAR 全局数据
            │   - 驱动数据
            │
0x2001_0000 ├─ 堆空间 (Heap)
            │   (向上增长)
            │
0x2002_0000 └─ 结束
```

## 配置系统

所有配置集中在 `AUTOSAR/AUTOSAR_Cfg.h`：

- CAN ID 配置
- 波特率配置
- PDU 路由配置
- 信号配置
- DID 配置

## 调试系统

通过 DebugLog 模块输出调试信息：

- 系统启动日志
- 模块初始化日志
- CAN 收发日志
- 错误日志

输出接口：USART1 (PA9-TX, PA10-RX)，115200bps

## 扩展指南

### 添加新的 CAN ID

1. 在 `AUTOSAR_Cfg.h` 中添加 ID 定义
2. 在 `CanIf_TxPduConfig` 或 `CanIf_RxPduConfig` 中添加配置
3. 在 PduR 路由表中添加路由

### 添加新的 UDS 服务

1. 在 `Dcm.c` 中添加服务处理函数
2. 在 `Dcm_ProcessService()` 中添加分发
3. 更新 `Dcm_Config` 配置

### 添加新的信号

1. 在 `Com_SignalConfig` 中添加信号配置
2. 在应用层调用 `Com_SendSignal()` 或 `Com_ReceiveSignal()`
