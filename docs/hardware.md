# 硬件连接指南

本文档描述 STM32F407 AUTOSAR 项目的硬件连接方式。

## 开发板要求

### 主控芯片
- **型号**: STM32F407ZGT6
- **封装**: LQFP144
- **主频**: 168MHz
- **Flash**: 1MB
- **RAM**: 128KB

### 时钟配置
- **外部晶振**: 8MHz HSE
- **系统时钟**: 168MHz
- **APB1 时钟**: 42MHz (CAN 时钟源)
- **APB2 时钟**: 84MHz (USART 时钟源)

## 引脚分配

### CAN 接口

| 功能 | 引脚 | 模式 | 说明 |
|------|------|------|------|
| CAN1_RX | PA11 | 复用功能 AF9 | CAN 接收 |
| CAN1_TX | PA12 | 复用功能 AF9 | CAN 发送 |

**连接方式**:
```
STM32F407                    PCAN-USB
┌─────────┐                ┌─────────┐
│         │                │         │
│  PA11   │◄──────────────►│  CAN_H  │
│  (RX)   │                │         │
│         │                │         │
│  PA12   │◄──────────────►│  CAN_L  │
│  (TX)   │                │         │
│         │                │         │
│  GND    │◄──────────────►│  GND    │
│         │                │         │
└─────────┘                └─────────┘
```

**注意事项**:
- 使用双绞线连接 CAN_H 和 CAN_L
- 总线两端需要 120Ω 终端电阻
- 确保共地连接

### 串口调试

| 功能 | 引脚 | 模式 | 说明 |
|------|------|------|------|
| USART1_TX | PA9 | 复用功能 AF7 | 串口发送 |
| USART1_RX | PA10 | 复用功能 AF7 | 串口接收 |

**连接方式**:
```
STM32F407                    USB转串口
┌─────────┐                ┌─────────┐
│         │                │         │
│  PA9    │◄──────────────►│  RX     │
│  (TX)   │                │         │
│         │                │         │
│  PA10   │◄──────────────►│  TX     │
│  (RX)   │                │         │
│         │                │         │
│  GND    │◄──────────────►│  GND    │
│         │                │         │
└─────────┘                └─────────┘
```

**串口参数**:
- 波特率: 115200
- 数据位: 8
- 停止位: 1
- 校验: 无
- 流控: 无

### LED 指示灯

| LED | 引脚 | 模式 | 说明 |
|-----|------|------|------|
| LED0 | PE3 | 输出 | 系统运行指示 |
| LED1 | PE4 | 输出 | 预留 |
| LED2 | PG9 | 输出 | 心跳指示 |

**连接方式**:
```
STM32F407                    LED
┌─────────┐                ┌───────┐
│         │                │       │
│  PE3    │──────┳────────►│  LED0 │
│  (LED0) │      │         │       │
│         │      ┃         └───────┘
│  PE4    │──────┻────────►┌───────┐
│  (LED1) │                │       │
│         │                │  LED1 │
│  PG9    │───────────────►│       │
│  (LED2) │                └───────┘
│         │
│  GND    │◄──────────────── 限流电阻
│         │
└─────────┘
```

**注意事项**:
- 低电平点亮（输出 0 时 LED 亮）
- 建议串联 330Ω 限流电阻
- LED 阴极接 GPIO，阳极通过电阻接 VCC

### 调试接口

支持 SWD 调试：

| 功能 | 引脚 | 说明 |
|------|------|------|
| SWDIO | PA13 | SWD 数据 |
| SWCLK | PA14 | SWD 时钟 |
| NRST | NRST | 复位 |

**连接方式**:
```
STM32F407                    J-Link
┌─────────┐                ┌─────────┐
│         │                │         │
│  PA13   │◄──────────────►│  SWDIO  │
│ (SWDIO) │                │         │
│         │                │         │
│  PA14   │◄──────────────►│  SWCLK  │
│ (SWCLK) │                │         │
│         │                │         │
│  NRST   │◄──────────────►│  RST    │
│         │                │         │
│  3.3V   │◄──────────────►│  VTref  │
│         │                │         │
│  GND    │◄──────────────►│  GND    │
│         │                │         │
└─────────┘                └─────────┘
```

## 电源要求

### 供电电压
- **VDD**: 3.3V ± 5%
- **VDDA**: 3.3V ± 5% (模拟电源)
- **VREF+**: 3.3V (ADC 参考电压)

### 电流要求
- **正常工作**: 约 50-100mA
- **LED 点亮**: 每个 LED 约 10mA
- **CAN 收发**: 约 50mA (取决于总线负载)

**建议**: 使用 500mA 以上电源

## PCB 设计建议

### CAN 总线
- 使用双绞线
- 差分阻抗 120Ω
- 终端电阻放置在总线两端

### 电源
- 每个 VDD 引脚就近放置 100nF 去耦电容
- VCAP_1/VCAP_2 连接 2.2uF 电容到地
- VDDA 通过 LC 滤波与 VDD 隔离

### 晶振
- HSE 使用 8MHz 无源晶振
- 晶振尽量靠近芯片
- 匹配电容根据晶振规格选择

## 常见问题

### Q: CAN 通信不稳定
**A**: 检查以下事项：
- 终端电阻是否正确（两端各 120Ω）
- 波特率是否一致（500Kbps）
- 共地是否良好
- 线缆是否为双绞线

### Q: 串口乱码
**A**: 检查以下事项：
- 波特率是否为 115200
- 共地是否良好
- USB转串口模块是否稳定

### Q: LED 不亮
**A**: 检查以下事项：
- 是否为低电平点亮（代码输出 0）
- 限流电阻是否正确
- LED 极性是否正确

## 参考设计

- STM32F407ZGT6 主控
