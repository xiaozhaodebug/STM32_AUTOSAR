# 硬件外设规格说明

本文档详细说明 STM32_AUTOSAR 项目使用的硬件外设配置。

## 1. 主控芯片

| 参数 | 规格 |
|------|------|
| **型号** | STM32F407ZGT6 |
| **内核** | ARM Cortex-M4 |
| **主频** | 168 MHz |
| **Flash** | 1024 KB (1MB) |
| **SRAM** | 128 KB |
| **封装** | LQFP144 |
| **工作电压** | 3.3V |

## 2. 时钟系统

### 2.1 外部晶振

| 参数 | 规格 |
|------|------|
| **晶振类型** | HSE (高速外部晶振) |
| **频率** | 8 MHz |
| **用途** | 系统主时钟源 |
| **负载电容** | 建议 20pF |

### 2.2 时钟树配置

### 2.3 时钟参数

| 时钟源 | 频率 | 用途 |
|--------|------|------|
| SYSCLK | 168 MHz | CPU 系统时钟 |
| HCLK (AHB) | 168 MHz | AHB 总线时钟 |
| PCLK1 (APB1) | 42 MHz | APB1 外设时钟 (CAN1) |
| PCLK2 (APB2) | 84 MHz | APB2 外设时钟 (USART1) |

## 3. CAN 外设

### 3.1 CAN1 配置

| 参数 | 配置 |
|------|------|
| **外设** | CAN1 |
| **RX 引脚** | PA11 |
| **TX 引脚** | PA12 |
| **复用功能** | AF9 (GPIO_AF9_CAN1) |
| **波特率** | 500 Kbps |
| **时钟源** | APB1 (42 MHz) |

### 3.2 CAN 位时序配置

| 参数 | 值 | 说明 |
|------|-----|------|
| **Prescaler** | 6 | 预分频系数 |
| **Time Segment 1 (BS1)** | 11 | 时间段 1 |
| **Time Segment 2 (BS2)** | 2 | 时间段 2 |
| **SJW** | 1 | 同步跳转宽度 |
| **采样点** | 85.7% | (1+BS1)/(1+BS1+BS2) |

**波特率计算公式**:
```
波特率 = CAN时钟 / ((Prescaler) × (1 + BS1 + BS2))
500 Kbps = 42 MHz / (6 × 14) = 42 MHz / 84
```

### 3.3 CAN 硬件连接

```
STM32F407                    CAN收发器                    CAN总线
┌─────────┐                ┌─────────┐                ┌─────────┐
│         │                │         │                │         │
│  PA11   │◄──────────────►│   RX    │                │         │
│ (CAN1_RX)│                │         │                │         │
│         │                │  TJA    │◄──────────────►│  CAN_H  │
│  PA12   │◄──────────────►│  1050   │                │         │
│ (CAN1_TX)│                │  or     │                │         │
│         │                │SN65HVD23│◄──────────────►│  CAN_L  │
│  3.3V   │◄──────────────►│   VCC   │                │         │
│         │                │         │                │         │
│  GND    │◄──────────────►│   GND   │                │         │
│         │                │         │                │         │
└─────────┘                └─────────┘                └─────────┘
```

### 3.4 CAN 滤波器配置

- **滤波器 0**: 接收所有消息 (ID=0, Mask=0)
- **模式**: 32位掩码模式
- **FIFO**: FIFO0

## 4. USART 外设

### 4.1 USART1 配置

| 参数 | 配置 |
|------|------|
| **外设** | USART1 |
| **TX 引脚** | PA9 |
| **RX 引脚** | PA10 |
| **复用功能** | AF7 (GPIO_AF7_USART1) |
| **波特率** | 115200 bps |
| **数据位** | 8 |
| **停止位** | 1 |
| **校验** | 无 |
| **流控** | 无 |
| **时钟源** | APB2 (84 MHz) |

### 4.2 USART 波特率计算

```
波特率 = USART时钟 / (16 × BRR)
115200 = 84 MHz / (16 × 45.57)
BRR = 0x02D9 (729)
```

### 4.3 USART 硬件连接

```
STM32F407                    USB转串口模块
┌─────────┐                ┌─────────┐
│         │                │         │
│  PA9    │◄──────────────►│  RX     │
│(USART1_TX)│               │         │
│         │                │  CH340  │
│  PA10   │◄──────────────►│  TX     │
│(USART1_RX)│               │  or     │
│         │                │ CP2102  │
│  GND    │◄──────────────►│  GND    │
│         │                │         │
└─────────┘                └─────────┘
```

## 5. GPIO 外设 (LED)

### 5.1 LED 引脚配置

| LED | 引脚 | 模式 | 极性 | 默认状态 |
|-----|------|------|------|----------|
| **LED0** | PE3 | 推挽输出 | 低电平点亮 | 熄灭 |
| **LED1** | PE4 | 推挽输出 | 低电平点亮 | 熄灭 |
| **LED2** | PG9 | 推挽输出 | 低电平点亮 | 熄灭 |

### 5.2 LED 硬件连接

```
STM32F407              LED电路
┌─────────┐           ┌─────────┐
│         │           │         │
│  PE3    │──────────►│  LED0   │────┐
│ (LED0)  │           │         │    │
│         │           └─────────┘    │
│  PE4    │──────────►┌─────────┐    │
│ (LED1)  │           │  LED1   │────┤
│         │           │         │    │    ┌─────┐
│  PG9    │──────────►└─────────┘    └────┤330Ω │
│ (LED2)  │           ┌─────────┐         │电阻 │
│         │           │  LED2   │─────────┤     │
│         │           │         │         └─────┘
│         │           └─────────┘             │
│  VCC    │                                   │
│ (3.3V)  │◄─────────────────────────────────┘
│         │
└─────────┘
```

### 5.3 LED 驱动方式

- **低电平点亮**: GPIO输出 0 时 LED 亮
- **高电平熄灭**: GPIO输出 1 时 LED 灭
- **限流电阻**: 建议 220Ω ~ 470Ω

### 5.4 LED 功能定义

| LED | 功能 | 说明 |
|-----|------|------|
| LED0 (PE3) | 系统运行指示 | 启动时常亮 |
| LED1 (PE4) | 预留 | 用户自定义 |
| LED2 (PG9) | 心跳指示 | 每秒闪烁 |

## 6. 定时器外设

### 6.1 SysTick 配置

| 参数 | 配置 |
|------|------|
| **外设** | SysTick (内核定时器) |
| **时钟源** | HCLK (168 MHz) |
| **重载值** | 168000 - 1 |
| **中断周期** | 1 ms |
| **用途** | 系统时钟、任务调度 |

### 6.2 SysTick 计算

```
中断周期 = (Reload + 1) / HCLK
1ms = 168000 / 168 MHz
```

## 7. 中断配置

### 7.1 中断优先级

| 中断源 | 优先级 | 用途 |
|--------|--------|------|
| SysTick | 0 (最高) | 系统时钟 |
| CAN1_RX0 | 1 | CAN 接收 |
| USART1 | 2 | 串口通信 |

### 7.2 NVIC 配置

- **抢占优先级**: 4 位 (16 级)
- **子优先级**: 0 位
- **优先级分组**: Group 4

## 11. 硬件检查清单

### 11.1 必需组件

- [ ] STM32F407ZGT6 主控芯片
- [ ] 8MHz 外部晶振
- [ ] 3.3V 稳压电源
- [ ] CAN 收发器 (TJA1050 或 SN65HVD230)
- [ ] USB 转串口模块 (CH340 或 CP2102)
- [ ] 3 个 LED (带限流电阻)
- [ ] J-Link 或 ST-Link 调试器

### 11.2 可选组件

- [ ] 复位按钮
- [ ] 启动模式选择开关
- [ ] 电源指示灯
- [ ] 更多 GPIO 扩展接口

## 12. 注意事项

1. **电源**: 确保 3.3V 电源稳定，建议使用 LDO
2. **晶振**: HSE 晶振尽量靠近芯片，负载电容匹配
3. **CAN**: 总线两端需要 120Ω 终端电阻
4. **串口**: 确保 USB 转串口模块与 STM32 共地
5. **LED**: 注意极性和限流电阻，低电平点亮
